# Objetivos del Producto

## Objetivo general

Desarrollar una aplicación web de productividad personal que integre tareas, hábitos y objetivos en una experiencia unificada, permitiendo al usuario planificar su semana y hacer seguimiento de su progreso personal.

## Objetivos específicos

### O1 — Gestión de tareas
Proveer un sistema completo de gestión de tareas con prioridades, fechas límite, categorías y filtros, que permita al usuario organizar su trabajo diario de forma eficiente.

### O2 — Seguimiento de hábitos
Permitir al usuario definir hábitos, registrarlos diariamente y visualizar rachas (streaks) para fomentar la consistencia.

### O3 — Establecimiento de objetivos
Facilitar la creación de objetivos con tareas asociadas y seguimiento visual del progreso.

### O4 — Visión semanal
Ofrecer una vista semanal que integre tareas pendientes, hábitos del día y progreso de objetivos, proporcionando contexto y claridad.

### O5 — Experiencia de usuario excelente
Conseguir una interfaz limpia, rápida y responsive que funcione en desktop y móvil, con mínima fricción.

### O6 — Autenticación y datos seguros
Implementar autenticación segura (JWT + bcrypt) y proteger los datos personales del usuario, aunque sean de baja sensibilidad.

### O7 — API documentada
Exponer una API REST documentada con OpenAPI/Swagger que permita consumo por terceros o integración futura con clientes nativos.

### O8 — MVP validable en 3 meses
Tener una versión funcional (MVP) en un plazo máximo de 12 semanas desde el inicio del desarrollo, que cubra el ciclo básico de productividad personal.
