# Ronsel

> A personal productivity web application focused on tasks, habits, goals, and planning.

Ronsel helps you organize your daily life around three core pillars: what you need to do (tasks), what you want to build (habits), and where you're going (goals).

## Tech Stack

| Category | Technology |
|---|---|
| Frontend | React (Vite) |
| Backend | Node.js + Express.js |
| Database | PostgreSQL (Neon) |
| ORM | Prisma |
| Auth | JWT + bcrypt |
| Validation | Zod |
| Testing | Vitest + React Testing Library |
| API Docs | OpenAPI / Swagger |
| Deploy | Vercel (FE) · Railway (BE) · Neon (DB) |

## Commands

```bash

npm run build     # Build for production
npm run test      # Run tests

# Backend

npm run build     # Build for production
npm run lint      # Lint code
```
